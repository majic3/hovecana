About

Cascaded StyleSheet for horisontal and vertical navigation menus or tabs. Not depended on content ("Uordered List" tag only)

The aim of this changes to this fork is to make a range of nav systems for use with OOCSS

The styles are arranged in skins (by color) and by base styles by name. These are then applied to mark up using multiple classes. The eventual plan is to use this with OOCSS and tools like compass to create a really tight output/final css fr use with a site - at this stage your defining some rules in place of the skin rules (applying your skin) and applying the final styles to you code how you want to - Your semantic styles for your code - no excess rules neat compact (no-redundant) clean set of rules

.nav ul.hor, .nav ul.ver{position:absolute}
